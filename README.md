# Proyecto: La aproximación de Apophis en 2029

Proyecto del curso de **Mecánica Celeste** (2026-1) — Prof. Jorge I. Zuluaga

## Contenido

- `proyecto-del-curso-Apophis.ipynb`: Notebook principal con la simulación y el análisis del encuentro de Apophis con la Tierra en abril de 2029.

## Métodos utilizados

- Integrador Leapfrog
- `ncuerpos_solucion` (pymcel)
- REBOUND + NASA Horizons (IAS15)
- Problema de dos cuerpos (trayectoria hiperbólica)
